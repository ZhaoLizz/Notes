# 二叉查找树

* 每个结点的键都**大于左子树中任意结点的键,小于右子树中任意结点的键**  

## 基本实现

```java
public class BST<Key extends Comparable<Key>, Value> {
    private class Node {
        private Key key;
        private Value val;
        private Node left, right;
        private int N;

        public Node(Key key, Value val, int n) {
            this.key = key;
            this.val = val;
            N = n;
        }
    }

    private Node root;

    public int size() {
        return size(root);
    }

    private int size(Node x) {
        if (x == null) return 0;
        else return x.N;
    }
}

```

## `get()`和`put()`方法

```java
    public Value get(Key key) {
        return get(root, key);
    }

    private Value get(Node x, Key key) {
        if (x == null)
            return null;
        int cmp = key.compareTo(x.key);

        //key < x.key
        if (cmp < 0) return get(x.left, key);
        else if (cmp > 0) return get(x.right, key);
        else return x.val;
    }

    public void put(Key key, Value val) {
        root = put(root, key, val);
    }

    private Node put(Node x, Key key, Value val) {
        //如果key存在于以x为根节点的子树中,则更新值
        //否则将新节点插入到该子树中
        if (x == null)
            return new Node(key, val, 1);

        int cmp = key.compareTo(x.key);
        if (cmp < 0) x.left = put(x.left, key, val);
        else if (cmp > 0) x.right = put(x.right, key, val);
        else x.val = val;

        x.N = size(x.left) + size(x.right) + 1;
        return x;
    }
```


## `max() min() floor() ceiling()`

```java
    public Key min() {
        return min(root).key;
    }

    private Node min(Node x) {
        if (x.left == null) return null;
        return min(x.left);
    }

    public Key max() {
        return max(root).key;
    }

    private Node max(Node x) {
        if (x.right == null) return null;
        return max(x.right);
    }

    //返回小于等于当前Key的树中的key
    public Key floor(Key key) {
        Node x = floor(root, key);
        if (x == null) return null;
        return x.key;
    }


    private Node floor(Node x, Key key) {
        if (x == null) return null;
        int cmp = key.compareTo(x.key);

        //如果相等,直接返回当前节点
        if (cmp == 0) return x;
        //如果key小于当前结点的key,向左子树递归
        if (cmp < 0) return floor(x.left, key);

        //else key大于当前结点的key,直接找当前结点的右子树
        Node t = floor(x.right, key);
        if (t != null) return t;
        else return x;
    }

```


## 选择操作 `select(), rank()`

```java
public Key select(int k) {
        return select(root, k).key;
    }

    /**
     * 返回键排名为k的结点
     * 排名从0开始计数
     */
    private Node select(Node x, int k) {
        if (x == null) return null;
        int t = size(x.left);

        if (t > k) return select(x.left, k);
        else if (t < k) return select(x.right, k - t - 1);
        else return x;
    }

    public int rank(Key key) {
        return rank(key, root);
    }

    /**
     * 返回给定键的排名
     * 排名从0开始计数
     */
    private int rank(Key key, Node x) {
        if (x == null) return 0;
        int cmp = key.compareTo(x.key);     //key和当前结点的key比较

        //key < 当前节点的key
        if (cmp < 0) return rank(key, x.left);
            //1是指当前结点(不是目标结点)
        else if (cmp > 0) return 1 + size(x.left) + rank(key, x.right);
        else return size(x.left);
    }
```


## 删除操作
* 在删除结点x后用它的**后继结点**填补它的位置.因为x有一个右子节点,所以它的后继结点就是其右子树中的最小结点
        1. 将指向即将被删除的结点的链接保存为t
        1. 将x指向它的后继结点min(t.right)
        1. 将x的右链接指向deleteMin(t.right)   (因为x拿出来做新的子树结点了,所以删除掉原来x所属的子树中本属于x的位置)
        1. 将x的左链接设为t.left

```java
    private Node deleteMin(Node x) {
        if(x.left == null)
            return x.right;

        x.left = deleteMin(x.left);
        x.N = size(x.left) + size(x.right) + 1;
        return x;
    }

    public void deleteMin() {
        root = deleteMin(root);
    }

    private Node deleteMax(Node x) {
        if (x.right == null)
            return x.left;
        x.right = deleteMax(x.right);
        x.N = size(x.left) + size(x.right) + 1;
        return x;
    }

    private Node delete(Node x, Key key) {
        if (x == null)
            return null;
        int cmp = key.compareTo(x.key);

        if (cmp < 0)
            x.left = delete(x.left, key);
        else if (cmp > 0)
            x.right = delete(x.right, key);
            //命中结点
        else {
            if (x.right == null)
                return x.left;
            if (x.left == null)
                return x.right;

            Node t = x;
            x = min(t.right);
            x.right = deleteMin(t.right);
            x.left = t.left;
        }

        x.N = size(x.left) + size(x.right) + 1;
        return x;
    }

    public void delete(Key key) {
        root = delete(root, key);
    }

```

## 遍历和范围查找
    * 用**中序遍历**的方式即可按顺序遍历
    * 范围查找: 将所有落在给定范围以内的键加入一个队列Queue并跳过那些不可能含有所查找键的子树
        * 只要左边界小于当前结点,就一直向坐下走

```java
private void keys(Node x, Queue<Key> queue, Key lo, Key hi) {
        if (x == null)
            return;
        int cmplo = lo.compareTo(x.key);
        int cmphi = hi.compareTo(x.key);
        if (cmplo < 0)
            keys(x.left, queue, lo, hi);
        if (cmphi <= 0 && cmphi >= 0)
            queue.add(x.key);
        if (cmphi > 0)
            keys(x.right, queue, lo, hi);
    }

```
